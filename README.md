## Quick Start

```bash
# 1. Setup Blockchain
cd BlockChain && forge install && forge build

# 2. Setup Backend  
cd ../Backend && npm install && npm run dev

# 3. Setup Frontend (new terminal)
cd ../FrontEnd && npm install && npm run dev
```

Backend: `http://localhost:5000`
Frontend: `http://localhost:5173`
